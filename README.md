# Action Movie Log

This project is a simple front-end movie tracker built using HTML, CSS, and JavaScript.  
Users can add action movies to a list, search through the list, mark movies as watched, and remove movies they no longer want to track.

## Features
- Add movies to the list
- Remove movies
- Mark movies as watched
- Search movies instantly
- Hover interactions and animations
- Responsive layout for different screen sizes
- Background image for a cinematic look

## Technologies Used
- HTML
- CSS
- JavaScript
